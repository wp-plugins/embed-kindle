Embed Kindle e-books viewer Kindle Reader by short tag

===================================================================

== Description ==

This is a plugin to provide a short tag to embed Amazon's e-book viewer widget Kindle Reader in your posts.

Usage:

[embed_kindle asin="B004UW29IK"]

You may set your Amazon affiliate ID on admin setting.

PHP 5.3 or above required.

The admin menu is internationalized, currently supports English and Japanese (translation to your language is welcome).

== Installation ==

This section describes how to install the plugin and get it working.

1. Simply download the plug-in and manage it in your Wordpress Admin panel, under the Settings option. 
2. Unzip the downloaded package and upload the extracted folder into your Wordpress plugins folder.
3. Log into your Wordpress admin panel
4. Go to Plug-ins and Activate it
